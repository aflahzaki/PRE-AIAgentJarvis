## 🔄 Loop Interaktif

Program akan terus loop sampai Anda ketik `exit`:

```
Program start
    ↓
Welcome & instructions
    ↓
Input loop:
  1. Tunggu user input
  2. Proses dengan agent
  3. Tampilkan output
  4. Loop ke step 1
    ↓
User ketik 'exit'
    ↓
Program stop
```

## 🆘 Troubleshooting

### Q: Program tidak merespons
**A:** Tunggu 5-10 detik (API processing). Jika tetap, Ctrl+C dan restart.

### Q: Error "API key invalid"
**A:** Check API key di jarvis_coder.py line 14. Restart program.

### Q: Tidak muncul prompt input
**A:** Program mungkin stuck. Ctrl+C dan jalankan ulang.

### Q: File tidak ditemukan
**A:** Pastikan file ada di working directory. Gunakan `tools` untuk list files.
